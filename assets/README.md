Place app assets here. Expo expects:

- `icon.png` – app icon (1024×1024 recommended)
- `splash.png` – splash screen image
- `adaptive-icon.png` – Android adaptive icon foreground (1024×1024)

You can generate these with `npx expo prebuild` and replace them, or use the default assets from a new Expo project (`npx create-expo-app` and copy from there).

## Folders

- **logos/** – Brand logos, marks, wordmarks (e.g. SVG, PNG).
- **images/** – In-app imagery (photos, illustrations, backgrounds).
